- **分层启动**：`__main__.py` 作为统一入口，负责组装 `Settings`、`LLMClient`、`ToolRegistry` 及 `Agent` 核心，并启动 `CLI` 交互层。
- **契约驱动**：`Agent` 层通过 `LLMClient` 抽象进行异步 API 调用，并通过 `ToolRegistry` 动态获取工具定义；所有工具遵循 `Tool` 基类契约。
- **安全边界**：在对话循环中根据工具的 `danger_level` 属性实施拦截，高危操作（如 Shell 执行）必须经由 `CLI` 层回调获取用户显式授权。
- **配置隔离**：利用 `pydantic-settings` 统一管理环境变量，确保敏感信息与业务逻辑解耦，支持 `.env` 文件热加载。
