- **分层启动**：`__main__.py` 作为统一入口，按序实例化 `Settings`、`LLMClient`、`ToolRegistry`、`Agent` 及 `SessionManager`，最终注入 `CLI` 层启动 REPL。
- **契约驱动**：`Agent` 核心通过 `LLMClient` 抽象进行异步 API 调用，并通过 `ToolRegistry` 动态获取遵循 `Tool` 基类契约的工具定义。
- **安全边界**：在对话循环中根据工具的 `danger_level` 属性实施拦截，高危操作（如 Shell 执行）必须经由 `CLI` 层回调获取用户显式授权。
- **状态持久化**：利用 `SessionManager` 实现原子写策略，将不含 System Prompt 的对话历史持久化至 `.genesis/sessions/`，支持跨进程恢复。
