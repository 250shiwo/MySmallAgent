- **启动与配置**：`__main__.py` 作为入口，利用 `config.py` (基于 `pydantic-settings`) 加载 `.env` 环境变量并初始化全局设置。
- **核心循环**：`agent.py` 维护对话历史，驱动 LLM 交互循环；根据工具的 `danger_level` 动态触发 CLI 层的二次确认流程。
- **LLM 适配**：`llm.py` 封装 `AsyncOpenAI` 客户端，统一处理异步聊天请求与工具定义注入。
- **工具系统**：`tools/` 子包通过 `ToolRegistry` 集中管理继承自 `Tool` 基类的具体实现（如文件读写、Shell执行），并自动转换为 OpenAI API 格式。
- **交互界面**：`cli.py` 结合 `prompt_toolkit` 与 `rich` 构建 REPL，负责用户输入解析、状态渲染及危险操作的交互式确认。
- **会话持久化**：`session.py` 提供原子写策略的 JSON 存储，支持按 ID 前缀模糊匹配恢复历史会话。
